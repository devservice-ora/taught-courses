## 💻 Core Programming & Data Structures

### CS 102: Introduction to C++
*   **Focus:** Core foundational concepts of structured programming using C++. Cover variables, control flow, functions, arrays, and basic object-oriented principles.
*   **Documentation:** Active across Fall 2018, Fall 2019, Summer 2020, and Summer 2026 tracks.

### CS 116: C++ Programming
*   **Focus:** Advanced object-oriented design in C++, memory management, pointers, class hierarchies, polymorphism, and file I/O operations.
*   **Documentation:** Active across Spring 2018 tracks.

### CS 124: Data Structures
*   **Focus:** Abstract Data Types (ADTs), stacks, queues, linked lists, trees, graphs, sorting/searching algorithms, and algorithmic complexity analysis.
*   **Documentation:** Includes specialized lab manuals; active across Fall 2021, Fall 2024, and Spring 2025 tracks.

### CS 113: Discrete Structures
*   **Focus:** Mathematical foundations of computer science, including propositional logic, set theory, combinatorics, graph theory, and state machine simulations.
*   **Documentation:** Active across Spring 2020, Spring 2021, and Spring 2023 tracks.

## 🛡️ Systems Administration, Cisco Academy & Network Security

### CNET 146: Introduction to UNIX-Linux
*   **Focus:** Operating system fundamentals, user management, filesystem navigation, permissions, and command-line utilities.
*   **Documentation:** Active across Fall 2016 and Fall 2017 tracks.

### CNET 147: UNIX-Linux Shell Scripting
*   **Focus:** Automation architectures, Bash programming, text processing commands (sed, awk), and system administration scripts.
*   **Documentation:** Sustained summer track instruction spanning 2020, 2022, 2023, 2024, and 2025.

### CNET 170: Network Security
*   **Focus:** Comprehensive perimeter defense, cryptographic systems, firewall configurations, intrusion prevention, and operational security frameworks.
*   **Documentation:** Active across Fall 2017 and Fall 2018 tracks.

### CNET 172: CCNA Security
*   **Focus:** Cisco Academy curriculum mapping to associate-level network security certifications, implementing secure routing/switching infrastructures, and VPN technologies.
*   **Documentation:** Active across Fall 2016 and Fall 2017 tracks.

## 🏢 Legacy Enterprise Software & Distributed Environments

### CS 104A: Introduction to Visual Basic .NET
*   **Focus:** Event-driven programming model basics, managed code frameworks, desktop GUI application creation, and object-oriented architectures within .NET.
*   **Documentation:** Historical tracking across 2008, 2009, 2010, and 2012 tracks.

### CS 104B: Advanced .NET Programming
*   **Focus:** Multi-tier software architecture, persistent database connections (ADO.NET), and custom control components.
*   **Documentation:** Active across Fall 2007 tracks.

### CS 104D: .NET XML Web Services
*   **Focus:** Distributed service architectures, SOAP, WSDL protocols, message security, and service-oriented enterprise integration.
*   **Documentation:** Active across Spring 2007 and Spring 2008 tracks.

### CS 126: Internet Security Programming
*   **Focus:** Secure coding principles, auditing code for injection and overflow vulnerabilities, and implementing robust protocol defenses.
*   **Documentation:** Active across Fall 2004 tracks.

### CS 173: Java EE and EJB
*   **Focus:** Multi-tier corporate platform engineering, Enterprise JavaBeans (EJBs), component-based architectures, and servlet containers.
*   **Documentation:** Active across Spring 2012 tracks.
