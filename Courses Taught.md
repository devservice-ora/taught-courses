## 🏛️ Ohlone College Coursework

1. **CS/CNET-101 Introduction to Computers and Information Technology** An introduction to computer hardware, software, operating systems, and networking concepts, focusing on practical digital literacy and corporate IT foundations.

2. **CS-102 Introduction to C++ Programming** Introduction to computer programming using the C++ language, covering control structures, functions, arrays, and the basics of structured program design.

3. **CS-104A Introduction to .NET Programming** Foundational course on Microsoft's .NET framework, introducing managed code development, event-driven architecture, and UI basics.

4. **CS-104B Advanced .NET Programming** Advanced applications within the .NET ecosystem, exploring deep object-oriented concepts, multithreading, and advanced software design patterns.

5. **CS-104C ASP.NET** Focuses on building dynamic, data-driven web applications and enterprise-level websites using Microsoft’s ASP.NET technology stack.

6. **CS-104D Web Services** Explores the architecture, development, and deployment of distributed, service-oriented systems using SOAP, REST, and XML-based web protocols.

7. **CS-113 Discrete Structures** Mathematical foundations of computer science, including logic, set theory, combinatorics, graph theory, and recurrence relations crucial for algorithmic analysis.

8. **CS-116 Object-Oriented Programming using C++** Intermediate C++ programming focusing on object-oriented paradigms, including encapsulation, inheritance, polymorphism, and memory management.

9. **CS-118 Introduction to Assembly Language Programming** Low-level programming focusing on machine architecture, register allocation, memory addressing models, and direct hardware interfacing.

10. **CS-122 C# Programming** Comprehensive study of the C# programming language, emphasizing modern syntax, object-oriented concepts, exception handling, and collection types.

11. **CS-124 C++ Programming and Data Structures** Advanced data structures—such as linked lists, stacks, queues, trees, and graphs—implemented in C++ alongside algorithmic analysis and sorting techniques.

12. **CS-126 Security Programming** Focuses on secure coding principles, identifying vulnerabilities like buffer overflows and injection attacks, and implementing defensive software patterns.

13. **CS/CNET-146 Introduction to UNIX/Linux** An introduction to the UNIX/Linux operating system environment, covering file systems, navigation, command-line utilities, and user permissions.

14. **CS/CNET-147 Shell Programming** Advanced command-line automation and shell scripting (e.g., Bash) for parsing data, managing system resources, and streamlining IT operations.

15. **CNET-151 Systems and Network Administration** Practical skills required to manage server operating systems, handle user administration, implement network protocols, and maintain system uptime.

16. **CS/CNET-152 Data Communications** Covers the concepts and physical/logical layers of data communications, networking hardware, industry protocols, and structural network design.

17. **CS-175 JavaScript for Web Development** Client-side web scripting focusing on dynamic DOM manipulation, event handling, asynchronous actions, and modern front-end concepts.

18. **CS-178 XML** Structured data management using eXtensible Markup Language, focusing on document validation (DTDs/Schemas), parsing, and data transformation frameworks.

19. **CS-201 Special Project with NASA/CREST** Advanced student research and project-based coursework collaborating on real-world systems engineering and data processing tasks with NASA/CREST initiatives.
